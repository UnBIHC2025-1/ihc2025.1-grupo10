Ferramentas de Acessibilidade

Para auxiliar e facilitar o desenvolvimento de soluções acessíveis, foram elencadas ferramentas de acessibilidade abaixo.

  • Framework de Testes de Acessibilidade

    O Framework de Testes de Acessibilidade é um conjunto de ferramentas e bibliotecas que ajudam a testar a acessibilidade de aplicativos Android. Ele fornece uma estrutura para criar testes automatizados de acessibilidade e garantir que os aplicativos sejam acessíveis para todos os usuários, incluindo aqueles com deficiências.

  • Wave - web Accessibility Evaluation Tool

    Wave é uma ferramenta de avaliação de acessibilidade web que ajuda a identificar problemas de acessibilidade em páginas web. Ela verifica a acessibilidade de uma página web, fornecendo uma lista de problemas encontrados e sugestões para melhorar a acessibilidade.

  • DynoVisual Sitemap Generator

    DynoVisual Sitemap Generator é uma ferramenta que ajuda a criar mapas do site, permitindo que os usuários naveguem pelo site e encontrem facilmente as informações que precisam.

  • JAWS - leitor de telas para Windows

    JAWS é um leitor de telas para Windows que ajuda a navegar pelo sistema operacional e acessar todas as funcionalidades do sistema.

  • ACHECKS - Caixa de ferramentas para acessibilidade

    ACHECKS é uma caixa de ferramentas para acessibilidade que ajuda a testar a acessibilidade de aplicativos e sites.

  • AXE - Accessibility engine for automated Web UI testing

    AXE é uma ferramenta de teste de acessibilidade para aplicativos web que ajuda a identificar problemas de acessibilidade e melhorar a acessibilidade dos aplicativos.

  • Lighthouse - Google - Auditar acessibilidade

    Lighthouse é uma ferramenta de auditoria de acessibilidade para aplicativos web que ajuda a identificar problemas de acessibilidade e melhorar a acessibilidade dos aplicativos.

  • WebAim - Contrast Checker

    WebAim é uma ferramenta que ajuda a verificar a contraste de cores em páginas web, garantindo que os usuários possam ver o conteúdo de forma clara e legível.

  • Ases

    Ases é uma ferramenta brasileira de auditoria de acessibilidade para aplicativos web que ajuda a identificar problemas de acessibilidade e melhorar a acessibilidade dos aplicativos.

  • Accessibility Insights

    Accessibility Insights é uma ferramenta de auditoria de acessibilidade para aplicativos web que ajuda a identificar problemas de acessibilidade e melhorar a acessibilidade dos aplicativos.

  • Stark

    Stark é uma ferramenta de checagem de acessibilidade para aplicativos web que a partir de integrações que vão desde o processo de design até o desenvolvimento, ajuda a garantir que o aplicativo seja acessível para todos os usuários.

  • Jest-Axe

    Jest-axe é uma ferramenta de testes automatizados para componentes renderizados que ajuda a buscar violações de acessibilidade

  • Pa11y

    Pa11y é uma ferramenta em linha de comando que carrega páginas web a destaca possíveis problemas de acessibilidade.
